# Instalação de programas .deb (Debian, Ubuntu e derivados)

Baixar o arquivo.deb 

Instalar
-> dpkg -i arquivo.deb 

Listar 
-> dpkg -l 

Remover 
-> dpkg -r nome_do_pacote

Ajuda 
-> dpkg --help 

# Instalação de pacotes (Debian) 
apt-get update
apt-get install nome_do_pacote
apt-get remove nome_do_pacote
